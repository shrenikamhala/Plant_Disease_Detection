
# 🌿 Plant Disease Detection System

## 📌 Description
This project is a Machine Learning-based system that detects plant diseases from leaf images.  
It helps farmers and users identify plant diseases quickly and take necessary actions to improve crop health.

---

## 🚀 Features
- Detect plant diseases using image input
- Fast and accurate prediction
- User-friendly interface
- Supports multiple plant types
- Integrated chatbot support (if applicable)

---

## 🛠 Technologies Used
- Python
- Machine Learning
- TensorFlow / Keras
- OpenCV
- React (Frontend)
- Node.js (if used)
